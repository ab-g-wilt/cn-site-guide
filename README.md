# cn-site-guide

本仓库用于归档和发布多个独立 HTML 页面。  
这些页面内容涉及各类中文网站的使用说明、浏览指引或信息整理，以结构化、可离线查阅的静态页面形式呈现。

## 项目简介

- 本仓库不针对任何特定域名或具体网站。
- 所有页面均为独立 HTML 文件，可直接在浏览器中打开。
- 内容以文档整理、操作指引、信息汇总为主，不包含商业推广或诱导性内容。

## 目录说明

```
cn-site-guide/
├── pages/          # 存放所有独立 HTML 页面
├── assets/         # 公共资源文件（样式、图片、脚本等）
├── index.html      # 仓库首页或页面导航（可选）
└── README.md       # 本文件
```

- `pages/` 目录下每个 HTML 文件对应一个独立主题或站点说明。
- `assets/` 用于存放页面间共享的静态资源，便于统一维护。

## 页面归档说明

- 所有页面内容均来自公开可获取的信息，整理后以 HTML 格式保存。
- 页面内不嵌入追踪代码、第三方广告或动态请求。
- 归档旨在提供稳定的离线阅读体验，便于长期参考。

- 不接受包含具体商业合作、推广链接或诱导点击的贡献。

## 使用方式

1. 克隆或下载本仓库到本地。
2. 直接打开 `pages/` 目录下的 HTML 文件即可浏览。
3. 也可将仓库部署到静态托管服务上，作为网页访问。
